Hotel & promotion data

Edit the JSON below and save. hotels: id, name, city, country, lat, lng, program ("FHR" or "THC"). promos: hotelId, type ("nth_night_free" | "pct_off" | "credit"), n / pct / amount, stayStart, stayEnd, bookBy (optional), minNights (optional), sample (optional flag). pointValues: per-program baseline ¢/pt (cpp), Amex MR transfer ratio (mr), and your mrBaseline — tune these to your own valuations. Saved to this browser; use Export for a backup file.

Quick add Amex hotels

Copy hotel names straight off the Amex Travel FHR / Hotel Collection pages — one per line, ideally as “Name, City”. Each is located via OpenStreetMap Nominatim (about 1 per second) and added to the map. Lines that can't be located stay in the box so you can fix and retry.
